# 项目总览 — 盲盒去哪

> 最后更新：2026-06-27

---

## 一句话简介

「盲盒去哪」是一个基于高德地图的吃喝玩乐盲盒推荐工具，通过随机盲盒方式为用户推荐身边的美食、游玩、休闲去处，解决选择困难。

---

## 核心功能

- 🎲 **随机盲盒**：按品类（美食/游玩/休闲/全能）随机推荐周边店铺
- 🗺️ **地图集成**：基于高德地图 Web JS API 2.0 实现定位、搜索、导航
- 🤖 **AI 智能推荐**：接入 DeepSeek API 解析用户自然语言需求，精准推荐
- 📍 **距离筛选**：支持 1km / 3km / 5km / 不限 等多档距离筛选
- 💾 **收藏与历史**：本地存储收藏列表和开盒历史（最多 10 条）
- 🎨 **多款皮肤**：经典/可爱/简约/彩虹/渐变/霓虹/复古/水晶 共 8 款皮肤
- ❤️ **治愈文案**：每次开盒附带随机治愈语录
