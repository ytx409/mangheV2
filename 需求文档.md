# 需求文档 — 盲盒去哪

> 最后更新：2026-06-27

---

## 一、功能需求

### 1.1 核心功能：随机盲盒推荐
- 用户选择品类（美食/游玩/休闲/全能），点击「开盲盒」
- 系统根据用户位置 + 距离筛选 + 品类，随机推荐一个周边店铺
- 展示店铺名称、评分、人均价格、距离、导航链接

### 1.2 AI 智能推荐
- 用户用自然语言描述需求（如「想吃火锅，人均100左右」）
- 调用 DeepSeek API 解析意图，提取品类、关键词、氛围偏好
- 结合高德地图 POI 搜索返回精准推荐

### 1.3 城市切换与定位
- 支持热门城市列表选择
- 支持地址搜索（高德输入提示）
- 支持浏览器 GPS 定位

### 1.4 距离筛选
- 1公里以内 / 1~3公里 / 3~5公里 / 不限（5公里以内）

### 1.5 收藏与历史
- 收藏喜欢的店铺（localStorage）
- 查看开盒历史（最近 10 条）
- 历史记录去重

### 1.6 皮肤系统
- 8 款盲盒皮肤：经典/可爱/简约/彩虹/渐变/霓虹/复古/水晶
- 皮肤切换实时预览

### 1.7 结果分享
- 支持 Web Share API 分享
- 降级方案：复制到剪贴板

---

## 二、非功能需求

- 移动端优先，响应式设计
- 首次加载 < 3 秒
- 离线可查看收藏和历史
- 中文界面，治愈系 UI 风格
