# 踩坑日志 — 盲盒去哪

---

## 2026-06-27 — 皮肤 CSS 类缺失导致切换无样式

### 场景
用户切换到渐变/霓虹/复古/水晶皮肤时，盲盒卡片无样式。

### 错误表现
useBlindboxSkin() 返回 skinClass 为 lindbox-card-gradient 等，但 globals.css 中未定义对应 CSS 类。

### 根因分析
skin-config.ts 和 use-blindbox.ts 定义了 8 款皮肤，但 globals.css 只写了 4 款的 CSS。

### 修复方案
在 globals.css 中补充 lindbox-card-gradient、lindbox-card-neon、lindbox-card-retro、lindbox-card-crystal 四个类的样式定义。

### 预防措施
添加新皮肤时，确保同时更新 skin-config.ts 和 globals.css。
